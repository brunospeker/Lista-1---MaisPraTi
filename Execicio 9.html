Exercício 9

Implemente um programa que exibe uma contagem regressiva de 10 até 1 no console utilizando um loop for.

Entradas

Saidas

O resultado aparecerá aqui.

Código-Fonte

Copiar 

function contagemRegressiva() {
    let e1 = document.getElementById("entrada1").value;
    for(let c = e1; c > 0; c--){
        document.getElementById("resultado").innerText = "O número atual é "+c+"! Verifique no console para enxergar a contagem regressiva.";
        console.log(c);
    }
}
