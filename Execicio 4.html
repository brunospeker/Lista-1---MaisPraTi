Exercício 4

Crie um menu interativo no console que oferece ao usuário a escolha de três opções. Utilize switch-case para implementar a lógica de cada opção selecionada.

Entradas

Saidas

O resultado aparecerá aqui. Por favor, escolha uma das entradas.

Código-Fonte

Copiar 

function escolhaUm() {
    let e1 = document.querySelector('input[name="radio1"]:checked');
    
    switch (e1.value) {
        case "1":
            resultado.innerText = "Você escolheu a opção 1!";
            console.log("Você escolheu a opção 1!")

            resultado.innerText = "Você escolheu a opção 2!";
